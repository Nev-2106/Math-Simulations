# 📊 Math Simulations

A Python-based project for exploring and understanding advanced mathematical concepts through simulations, with a long-term goal of applying these models to **lithography computation and semiconductor systems**.

---

## 🚀 Overview

This repository is my experimental workspace where I:

- Learn mathematical concepts by implementing them computationally
- Translate equations into simulations and visual models
- Build intuition for complex systems used in physics and engineering
- Prepare for applying these methods to **real-world lithography and semiconductor problems**

> **Simulate → Understand → Apply**

---

## 🧠 Core Areas of Study

This project goes beyond basic ODEs and expands into key areas of modern science:

### Differential Equations
- Ordinary Differential Equations (ODEs)
- Partial Differential Equations (PDEs)
- Systems of coupled equations
- Boundary and initial value problems

### Mathematical Physics
- Wave equations
- Heat/diffusion equations
- Schrödinger equation (planned)
- Maxwell’s equations (planned)

### Transform Methods
- Fourier Series
- Fourier Transform
- Fast Fourier Transform (FFT)
- Laplace Transform

### Linear Algebra & Systems
- Eigenvalues and eigenvectors
- Matrix systems
- Stability analysis

### Dynamical Systems
- Phase space analysis
- Vector fields
- Chaos and nonlinear systems

### Numerical Methods
- Euler method
- Runge-Kutta methods
- Finite difference methods
- Finite element methods (planned)
